🧠 Reverse KPT – The Opposite of Type 1!

In Type 1 verbs, the 3rd person is strong and other forms are weak. In Types 3 (-lla/-llä), 4, and 6, it's reversed: the basic form is weak, and all personal forms are strong.

📌 Which Types Get Reverse KPT:

Type 3: only -lla/-llä endings Type 4: all verbs Type 6: all verbs Type 3 -nna/-nnä: no KPT
