🧠 Which Case for the Object?

The SVO sentence is Finnish's most common pattern. The object can be in partitive, nominative, or genitive — the rule depends on the context. Partitive always wins if there's a reason for it!

PARTITIVE kirjaa negative / ongoing / uncountable
|
NOMINATIVE kirja command / passive / must
|
GENITIVE kirjan finished / countable (default)
RULE:
Negative?
→
PARTITIVE
always!
En osta kirjaa. I don't buy a book. (negative → partitive)
Osta kirja! Buy the book! (command → nominative)
Luin kirjan. I read the book. (finished → genitive)
Luen kirjaa. I'm reading a book. (ongoing → partitive)

📌 Decision Order (partitive wins at each step):

1. Negative? → Partitive 2. Uncountable? → Partitive 3. Partitive verb? → Partitive 4. Command? → Nominative 5. Passive? → Nominative 6. Must (täytyy)? → Nominative 7. Default → Genitive

📋 Object Case Examples

En juo kahvia. partitive I don't drink coffee. (negative)
Otan kahvia. partitive I take some coffee. (uncountable)
Luen kirjaa. partitive I'm reading a book. (ongoing)
Opiskelen suomea. partitive I study Finnish. (partitive verb)
Osta kirja! nominative Buy the book! (command)
Täytyy ostaa kirja. nominative Must buy a book. (necessive)
Luin kirjan. genitive I read the book. (finished)
Ostan auton. genitive I buy a car. (one whole item)
