🔢 How to Form Ordinal Numbers

Add -s to the counting number (with some exceptions).

NUMBER neljä four
+
ENDING -s ordinal suffix
=
ORDINAL neljäs fourth
DATES:
ordinal
+
month + -ta
=
kahdeksas kesäkuuta = 8th of June
Ends in -n → drop n → kahdeksan → kahdeksas (8th) drop -n, then add -s
Ends in -si → drop i → viisi → viides (5th) stem changes: viisi → viide-
Irregular: 1st, 2nd, 3rd → ensimmäinen / eka, toinen / toka, kolmas must memorise these!

📋 Ordinal Numbers 1–10

1st ensimmäinen eka (spoken) first
2nd toinen toka (spoken) second
3rd kolmas third
4th neljäs fourth
5th viides fifth
6th kuudes sixth
7th seitsemäs seventh
8th kahdeksas eighth
9th yhdeksäs ninth
10th kymmenes tenth
